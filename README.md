# Microcontroladores - Prácticas
## Carlos Garza García

## Sofía Mijares Peña

En estas prácticas se trabajó con el microcontrolador **PIC16F887**, utilizando **MPLAB X IDE**, lenguaje **C**, compilador **XC8** y simulaciones en **Proteus**.

## Actividades realizadas

| Actividad | Nombre | Descripción |
|---|---|---|
| [Actividad en clase](./Actividad_Clase_Parpadeo_4_LEDs) | Parpadeo de 4 LEDs | Manejo básico de salidas digitales |
| [Actividad 1](./Actividad_1_Contador_6_Bits) | Contador binario de 6 bits | Conteo de 0 a 63 representado con LEDs |
| [Actividad 2](./Actividad_2_Caminata_8_Bits) | Secuencia tipo caminata de 8 bits | Desplazamiento secuencial de un LED |

## Estructura del repositorio

Cada actividad contiene su propia carpeta con:

- `README.md`: explicación de la práctica.
- `codigo/`: archivo en C utilizado para programar el PIC16F887.
- `evidencias/`: imágenes o videos de la simulación.
- `proteus/`: archivo de simulación realizado en Proteus.

---

## Objetivo general

Comprender el manejo de salidas digitales en el microcontrolador PIC16F887 mediante la configuración de puertos, escritura de valores binarios y control de LEDs.
